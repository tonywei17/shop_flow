## 使用方法

### ステップ1: ログインページにアクセス
```
http://localhost:3002/auth/login
```

### ステップ2: デモアカウントを選択
- 無料会員を体験したい場合: 「無料会員でログイン」をクリック
- プレミアム会員を体験したい場合: 「プレミアム会員でログイン」をクリック

### ステップ3: 自動ログイン
ボタンをクリックすると、自動的にログインして会員中心（マイページ）にリダイレクトされます。

### ステップ4: プラットフォームを体験
ログイン後、以下のページを自由に閲覧できます:
- マイページ (`/dashboard`)
- コース一覧 (`/courses`)
- コース詳細 (`/courses/[id]`)
- ビデオ学習 (`/courses/[id]/learn`)
- 活動一覧 (`/activities`)
- 通知センター (`/notifications`)

## 技術実装

### データ保存方法
デモアカウント情報は `localStorage` に保存されます:

```javascript
localStorage.setItem('demoUser', JSON.stringify({
  name: '山田太郎',
  email: 'demo-premium@example.com',
  memberType: 'premium',
  joinDate: '2024-06-15',
  coursesEnrolled: 5,
  qualifications: 2
}));
```

### データ読み取り
会員中心ページで `localStorage` からデータを読み取ります:

```javascript
useEffect(() => {
  const demoUserStr = localStorage.getItem('demoUser');
  if (demoUserStr) {
    const demoUser = JSON.parse(demoUserStr);
    setUser({
      name: demoUser.name,
      email: demoUser.email,
      membershipType: demoUser.memberType,
      // ... その他の情報
    });
  }
}, []);
```

## 注意事項

### セキュリティ
- ⚠️ これは**デモ専用**の機能です
- ⚠️ 本番環境では削除または無効化してください
- ⚠️ 実際の認証システムではありません

### データの永続性
- デモアカウントデータは `localStorage` に保存されます
- ブラウザのキャッシュをクリアするとデータが消えます
- ページをリロードしてもデータは保持されます

### ログアウト
現在、ログアウト機能は実装されていません。
デモアカウントを切り替えるには:
1. ブラウザの開発者ツールを開く
2. Application > Local Storage を選択
3. `demoUser` キーを削除
4. ページをリロード

## トラブルシューティング

### 問題: ログイン後、データが表示されない
**解決方法**:
1. ブラウザのコンソールを確認
2. `localStorage` に `demoUser` が保存されているか確認
3. ページをリロード

### 問題: ボタンをクリックしても反応しない
**解決方法**:
1. ブラウザのコンソールでエラーを確認
2. JavaScript が有効になっているか確認
3. ページをリロード

### 問題: 会員タイプが正しく表示されない
**解決方法**:
1. `localStorage` の `demoUser` データを確認
2. `memberType` が 'free' または 'premium' であることを確認
3. データを削除して再度ログイン
